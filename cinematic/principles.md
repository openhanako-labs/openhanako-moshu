# 电影化写作 — 重写原则

基于镜头分析结果，按以下原则重写。

## 六大原则

### 1. 景别切换代替过渡文字
不写"他看了看四周"，改用环境细节的层次展开。让镜头像电影一样从全景自然过渡到中景、近景、特写。

**Before:** 他看了看四周，茶馆里人不多。
**After:** 煤气灯下的木质吧台空了一半。角落里有个老者在打盹，茶杯歪在桌沿。

### 2. 节奏即镜头语言
句式长度即镜头节奏。
- 短句 = 快切 / 特写 / 紧张
- 长句 = 推轨 / 全景 / 舒缓

用句式长度替代时间副词（"突然""慢慢"），让读者从节奏本身感受情绪。

**Before:** 他慢慢地端起茶杯，慢慢喝了一口。
**After:** 杯沿抵住唇。一口。凉的。

### 3. 动作代替描述
内心独白转化为可见的动作、微表情、环境交互。

**Before:** 他知道这个人不对劲。
**After:** 他的手指在杯沿上停了三秒。

### 4. 蒙太奇连接
场景之间用意象 / 声音 / 颜色 / 动作完成过渡，而不是"与此同时"、"另一边"。

**Before:** 与此同时，在码头……
**After:** 茶馆里的雨声还没停。码头的缆绳断了。

### 5. 留白
镜头会说话。删掉"他知道""他感到"这类心理提示词，让读者从画面自己推断。

**Before:** 她感到一阵恐惧涌上心头。
**After:** 她后退了一步。鞋跟敲在木板上。

### 6. 声音设计
环境音 = 潜台词。用声音替代情绪词。

**Before:** 气氛很紧张。
**After:** 雨滴砸在瓦片上。没有人说话。茶水面上的浮沫正在散开。

## 改写力度分级

| 力度 | 范围 | 做法 |
|---|---|---|
| 轻度 | 局部调整（10-30%） | 替换过渡句、微调景别切换、删心理提示词 |
| 中度 | 段落重写（30-60%） | 重构段落节奏、增加景别层次、重写蒙太奇连接 |
| 重度 | 逐句重写（60%+） | 完全按分镜叙事重写，句式结构全部调整 |

## 禁忌

以下情况不要改：
- 不改情节走向
- 不增删场景
- 不改变叙事人称
- 不改变故事发生顺序
- 不替代润色/校对（错别字、语法仍由其他工具处理）
